How to Contribute

There are many ways to contribute to the Forgotten Future Project.

Play the game

The first way is: play the game! Feedback (through anonymous statistics or the old fashioned way) helps the contributors improve the game over time.

Back the game (Purchase/Subscribe)

The second way is: back the game! Forgotten Future is free to play for everyone. Everyone who donates appears on the master contribution list in the game credits.

Develop the game

The third way is: help develop the game! Forgotten Future is open-source, which means anyone can help work on the game visuals, code, and stage development. The source may also be forked as boiler plate for making your own games.
